# My_Cash

# A Transaction System made in C++

A console-based Mobile Banking Management System developed using C++.  
This project simulates the core functionalities of modern digital payment platforms such as bKash, Nagad, and Rocket.

The system provides secure account management, money transactions, bill payments, and transaction history tracking through a menu-driven interface.

---

## Features

### Authentication System
- User Registration
- Login System
- OTP Verification
- PIN Security
- Logout Functionality

### Account Management
- Update Account Information
- Remove/Delete Account
- Check Balance

### Transaction Features
- Cash In
- Cash Out
- Send Money
- Pay Bill
- Transaction History

---

## Technologies Used

- C++
- Object-Oriented Programming (OOP)
- File Handling
- Console-Based User Interface
